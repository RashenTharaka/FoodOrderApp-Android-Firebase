# User Guide

## App Flow

1. Open the app.
2. Tap the start/go button on the intro screen.
3. Browse food categories on the home page.
4. Tap a category to view foods under that category.
5. Tap a food item to open the detail page.
6. Select quantity using plus/minus buttons.
7. Tap the add button to add the item to the cart.
8. Open the cart from the bottom navigation.
9. Increase, decrease, or remove items from the cart.
10. Check item total, tax, delivery fee, and final total.

## Current Scope

This app is a food ordering UI project with Firebase data loading and cart management. It does not include real payments, login, order confirmation, delivery tracking, or admin management.
